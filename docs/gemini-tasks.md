# Gemini CSV作成タスク一覧

各タスクをGemini CLIに渡してCSVを作成する。
フォーマット仕様: `docs/csv-format.md` を参照。

完了したらこのファイルのチェックボックスを更新し、`docs/status.md` の該当セルを `csv` に更新する。

---

```
docs/csv-format.md の指示テンプレートに従って作業して。

注意: JABEE対応学科。別表Ⅰ・別表Ⅱ（p.46-51付近）からJABEEカラムも埋めること。

```
