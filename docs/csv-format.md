# 科目データCSVフォーマット

PDFの授業科目配当表からCSVを作成する際の仕様。
Gemini CLI / ChatGPT等でPDFを読み取り、このフォーマットでCSVを出力する。

## カラム定義

| カラム | 必須 | 説明 | 例 |
|--------|------|------|-----|
| 科目名 | ○ | 科目の正式名称 | 微分積分学Ⅰ |
| 単位数 | ○ | 数値 | 2 |
| 区分 | ○ | 工学基礎科目 / 専門教育科目 / 共通教育科目 | 工学基礎科目 |
| 第二区分 | ○ | 数学 / 自然科学技術 / 専門科目 / 英語科目 / 人間科学科目 等 | 数学 |
| 第三区分 | | より詳細な分類（任意） | 電気回路系 |
| 必選自 | ○ | 必修 / 選択 / 択一必修 / 自由 | 必修 |
| 配当年 | ○ | 1 / 2 / 全 / 1,2 / 2,3,4 等 | 1 |
| 配当期 | ○ | 前期 / 後期 / 前期／後期 / 通年 / 年次継続 | 前期 |
| 教職コード | | 空欄 = コードなし | 110代数 |
| 備考 | | 任意のメモ | 後期に再履修クラスを開講 |
| JABEE目標 | | C / C1 / D1 / D2 / E1 / E2 / F / B | D1 |
| JABEE電力 | | required / recommended / jabee-required | required |
| JABEEシステム | | 同上 | recommended |
| JABEEデバイス | | 同上 | required |

## Gemini/ChatGPTへの指示テンプレート

```
以下のPDFから授業科目配当表を読み取り、CSV形式で出力してください。
対象学科: {学科名}（{学科コード}）
入学年度: {年度}年度

出力ファイル名: data/csv/{学科コード小文字}-{年度下2桁}.csv

カラム: 科目名,単位数,区分,第二区分,第三区分,必選自,配当年,配当期,教職コード,備考,JABEE目標,JABEE電力,JABEEシステム,JABEEデバイス

ルール:
- 区分は「工学基礎科目」「専門教育科目」「共通教育科目」のいずれか
- 必選自は「必修」「選択」「択一必修」「自由」のいずれか
- 配当期は「前期」「後期」「前期／後期」「通年」「年次継続」のいずれか
- 教職コードは「コードなし」の場合は空欄
- JABEE関連カラムはJABEE対象学科のみ（EJのみ）。それ以外の学科は空欄
- JABEE分野の記号: ●=required, ○=recommended, ★=jabee-required
- 人間科学科目・英語科目も含めてすべての科目を出力すること
- 共通教育科目（人間科学・英語）は工学部全学科共通

また、以下の進級・卒業要件もテキストで出力してください:
- 2年進級条件
- 4年進級条件（必修科目数を含む）
- 卒業条件（区分別必要単位数）

最後に、docs/status.md の該当セルを「csv」に更新してください。
現在の対応状況: https://github.com/ykshio/sotsutan/blob/main/docs/status.md
```

## 進級・卒業要件

科目データとは別に、進級・卒業要件も以下の形式で提供する（テキストでOK）：

```
2年進級: 自由科目を除く修得単位合計24単位以上
4年進級: 自由科目除く104単位以上、人間科学10単位、英語6単位、必修XX科目
卒業: 人間科学16(技術者教養2,グローバル教養2含む), 英語8, 工学基礎18, 専門78, 任意4, 合計124
```

## ワークフロー

1. Geminiに上記テンプレートとPDFを渡してCSV + 要件テキストを出力
2. CSVを `data/csv/{コード}-{年度}.csv` に保存
3. `docs/status.md` の該当セルを `csv` に更新
4. Claude Codeに「`data/csv/xx-yy.csv` からデータを作って」と依頼
5. Claude CodeがTSファイル生成 → ビルド → PR作成
6. マージ後、`docs/status.md` を `done` に更新
